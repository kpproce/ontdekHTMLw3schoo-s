Opdr_1: Hoe kun je code van een website aanpassen (W3)?

  • Verander de tekt Hello World in Hallo jouw naam
    -> Klik op Groene knop Run om rechts te zien wat de code doet
  • Verander links in de de kleur Tomato voor de kleur Orange
    --> Klik op Groene knop Run om rechts te zien wat de code doet
    • - Probeer maar eens uit!
  • Kun je nog een blauwe tekst eronder zetten, zoals de eerste naar dan geen h1 maar h2 of h3
Zo ziet de website er uit
